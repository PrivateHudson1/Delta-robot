## Введение

В данном разделе описано аппаратное обеспечение промышленного дельта-робота: контроллер OpenCM 904, сервоприводы Dynamixel, система вакуума и безопасности. Эта информация необходима для правильного подключения, настройки и эксплуатации оборудования.

Контроллер OpenCM 904 является центральным устройством: он принимает команды с ПК по USB, управляет тремя сервоприводами Dynamixel по шине RS-485, контролирует аварийную кнопку, вакуумную присоску и зуммер. Все пины и интерфейсы зафиксированы в конфигурационных файлах и не требуют изменения при сборке.

**Общий принцип работы аппаратной части:**

1. ПК отправляет команду (например, `MOV 150 -200 -50`) через USB.
2. OpenCM 904 парсит команду, вычисляет углы для моторов (обратная кинематика).
3. По шине RS-485 контроллер передаёт целевые позиции моторам Dynamixel.
4. Моторы отрабатывают движение, возвращая текущие углы, ток и температуру.
5. OpenCM 904 постоянно мониторит аварийную кнопку и параметры моторов; при нарушении — отключает питание и включает зуммер.
6. Вакуумный клапан управляется отдельным цифровым выходом по команде с ПК.

## Пины OpenCM 904

| Пин | Назначение | Тип | Примечание |
|-----|------------|-----|------------|
| PA0 | ESTOP_PIN | INPUT_PULLUP | NC кнопка (LOW = авария) |
| PA1 | BUZZER_PIN | OUTPUT | активный HIGH |
| PA2 | VACUUM_PIN | OUTPUT | управляет реле/клапаном |
| PB10 | TX (Serial3) | OUTPUT | RS-485 передача |
| PB11 | RX (Serial3) | INPUT | RS-485 приём |
| PC12 | DXL_DIR_PIN | OUTPUT | направление RS-485 |

## Настройка пинов в коде (`configs/pins_config.h`)

```cpp
define DXL_SERIAL Serial3
define DXL_DIR_PIN 22
define ESTOP_PIN 2
define BUZZER_PIN 3
define VACUUM_PIN 4
```

## Моторы Dynamixel

| Параметр | Значение |
|----------|----------|
| Модель | AX-12A |
| Протокол | Version 1.0 |
| Скорость | до 1 000 000 бод |
| ID | 0 ~ 253 |
| Режим | Joint Mode (0° ~ 300°) / Wheel Mode |
| Напряжение питания | 9 ~ 12В (рекомендовано 11.1В) |
| Stall Torque | 1.5 N·m (при 12В) |
| Stall Current | 1.5 A |
| Скорость без нагрузки | 59 rpm (при 12В) |
| Ток в режиме ожидания | 50 mA |
| Позиционный сенсор | Потенциометр |
| Разрешение | 0.293° / pulse |
| Рабочая температура | -5°C ~ +70°C |
| Вес | 55 г |
| Gear Ratio | 254 : 1 |
| Материал шестерён | Инженерный пластик |
| Материал корпуса | Инженерный пластик |

**Важные особенности AX-12A:**
- Протокол 1.0 (не поддерживает расширенные функции Protocol 2.0)
- Управление через Compliance (Margin/Slope), не PID
- Физический уровень TTL (3-пиновый разъём: Data, Vcc, GND)
- Для груза 5 кг рекомендуется несколько моторов на степень свободы

## Питание

| Устройство | Напряжение | Ток (макс) |
|------------|------------|------------|
| OpenCM 904 | 5В (USB) | 500 мА |
| 3 × AX-12A | 11.1В | ~4.5 A |
| Вакуумный насос | 24В | 2 А |
| **Итого** | **11.1В / 24В** | **~7 А** |

**Рекомендация:** Два отдельных источника питания
- 11.1В Li-Po аккумулятор (или БП 12В, 5-7А) — для AX-12A
- 24В — для вакуумного насоса

## Аварийная кнопка
Параметр              | Значение
----------------------|-----------------------------------------------
Тип                   | нормально-замкнутая (NC) с фиксацией
Реакция               | немедленная остановка при разрыве цепи
Сброс                 | поворотом кнопки + команда RESET
