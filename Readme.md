# Delta Robot
Данный проект представляет собой полностью функционирующую систему управления промышленным дельта-роботом. Робот спроектирован для грузоподъемности **до 5 кг** и радиуса рабочей зоны **1 м**.

В основе системы лежит контроллер **OpenCM 9.04**, который управляет тремя сервоприводами **Dynamixel** (PRO или X серии) по протоколу Dynamixel 2.0 через шину RS-485. Связь с управляющим ПК осуществляется по USB (виртуальный COM-порт) со скоростью 115200 бод.

Прошивка на OpenCM написана на C++ (Arduino Framework) и включает:
- Кинематику дельта-робота (прямую и обратную)
- Планировщик S-образных траекторий
- Систему безопасности (мониторинг тока, температуры, E-stop)
- Двухрежимный протокол обмена (текстовый для отладки, бинарный с CRC для промышленной эксплуатации)

Управление с ПК реализовано на Python: консольный интерфейс позволяет отправлять команды движения, управлять вакуумом, запрашивать позицию и статус. В перспективе возможно добавление GUI и модуля компьютерного зрения (YOLO + OpenCV).

Проект полностью воспроизводим: любой разработчик может клонировать репозиторий, открыть в VS Code с PlatformIO, собрать прошивку одной кнопкой и загрузить её на контроллер. Все зависимости (библиотека Dynamixel2Arduino) устанавливаются автоматически

## Быстрый старт

1. Установить [VS Code](https://code.visualstudio.com/)
2. Установить расширение **PlatformIO IDE**
3. Открыть папку с проектом
4. Нажать `Ctrl+Alt+B` для сборки

## Структура

- `firmware/` — прошивка для OpenCM
- `configs/` — конфигурационные файлы
- `pc_software/` — ПО для ПК

## Зависимости

Все зависимости устанавливаются автоматически через PlatformIO.
