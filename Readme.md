# Delta Robot

Промышленный дельта-робот с грузоподъемностью до 5 кг

## Быстрый старт

1. Установить [VS Code](https://code.visualstudio.com/)
2. Установить расширение **PlatformIO IDE**
3. Открыть папку с проектом
4. Нажать `Ctrl+Alt+B` для сборки

## Структура

- `firmware/` — прошивка для OpenCM
- `configs/` — конфигурационные файлы
- `pc_software/` — ПО для ПК

## Зависимости

Все зависимости устанавливаются автоматически через PlatformIO.
